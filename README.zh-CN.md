# CHEMICAL-WORLD-VISUAL-GAME 🧪

一个基于 C# 与 [Godot](https://github.com/godotengine/godot) 的游戏

[English README](README.md)

## 简介

这是一款硬核游戏，难度较高，但值得一玩。游戏内容基于化学、工业与航天学。

本项目的目标是尽可能追求真实感，不使用魔法合成之类的设定。

## 如何运行

你需要安装 Godot。我们也推荐安装 Visual Studio 2022/2026。
