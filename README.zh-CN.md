# Chemical World Visual Game 🧪

一个以化学为主题，基于 C# 与 [Godot](https://github.com/godotengine/godot) 开发的游戏。

[English README](README.md)

## 简介

**Chemical World Visual Game** 是一款以化学、工业和航天工程为基础的硬核游戏。

游戏具有一定挑战性，适合喜欢真实系统和技术向玩法的玩家体验。

本项目的目标是尽可能追求真实感。游戏不会采用魔法合成之类的设定，而是尝试围绕真实的化学、工业和工程概念来构建玩法机制。

## 如何运行

运行本项目需要安装 [Godot](https://github.com/godotengine/godot)。

同时，我们也推荐安装 Visual Studio 2022 或 Visual Studio 2026，用于 C# 开发。
